La pipeline vera

Dal dominio all'output: ogni passo è tracciabile, recuperabile e ripetibile.

CLI → Spec → Context → Agents → Output

  1. CLI — raccoglie dominio, utenti, scope, regole, edge case
  2. SPEC — trasforma input in decisioni, acceptance, schema
  3. CONTEXT — diventa memoria locale del progetto
  4. AGENTS — istruzioni, prompt, skills, styleguide
  5. OUTPUT — app diverse, stesso ciclo di qualità
