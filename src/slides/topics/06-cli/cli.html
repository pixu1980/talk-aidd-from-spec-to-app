Wizardry without opaque magic

Each command has a precise responsibility. No floating prompts.

CLI commands

  • pnpm fill:context guided wizard, saved values, session resume
  • pnpm fill:context:status what is missing, which sections, next step
  • pnpm fill:context:next the next useful question without reopening all history
  • pnpm fill:context:doctor template, values, symlink, and generated output diagnostics
  • pnpm fill:context:add incremental granularity for scope, persona, need, feature
  • pnpm fill:context:export Markdown handoff for a new AI session or human review
