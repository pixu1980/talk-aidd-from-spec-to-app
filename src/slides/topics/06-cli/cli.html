Wizardry senza magia opaca

Ogni comando ha una responsabilità precisa. Nessun prompt volante.

Comandi CLI

  • pnpm fill:context wizard guidato, valori salvati, ripresa sessione
  • pnpm fill:context:status quanto manca, quali sezioni, prossimo passo
  • pnpm fill:context:next la prossima domanda utile, senza riaprire tutta la storia
  • pnpm fill:context:doctor diagnosi di template, valori, symlink e output generato
  • pnpm fill:context:add granularità incrementale per scope, persona, need, feature
  • pnpm fill:context:export handoff Markdown per nuova sessione AI o review umana
