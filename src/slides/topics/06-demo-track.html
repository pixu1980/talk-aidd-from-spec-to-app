Live sequence in 6 minutes

One complete cycle: from context to handoff, in real time.

Demo steps

  1. Show cli/.context/ as the source
  2. Run pnpm fill:context:status
  3. Add one need with pnpm fill:context:add:user-need
  4. Run pnpm fill:context:doctor
  5. Open .agents/skills/ and one concrete skill
  6. Close with pnpm fill:context:export
